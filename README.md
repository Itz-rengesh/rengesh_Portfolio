# rengesh-quant-portfolio

Repository Overview: RENGESH-QUANT
This repository houses the source code for RENGESH-QUANT, a high-performance, single-page professional portfolio tailored for the quantitative finance industry. The project serves as a centralized hub for displaying advanced algorithmic models, risk engines, and derivatives pricing frameworks.

Technical Architecture
The front-end is engineered for speed and precision, utilizing a minimalist tech stack to ensure low-latency rendering and high readability:

Framework: Built with Tailwind CSS v3, leveraging a utility-first approach for responsive grid layouts and rapid component styling.

Typography: Employs a dual-font system with Inter for clean headings and JetBrains Mono for technical/monospaced prose, reflecting a developer-centric environment.

Design System: Features a bespoke dark-mode theme (#0A0F14) with a custom neon-green accent (#00FF94) and a geometric grid-overlay system for a "terminal" aesthetic.

Key Functional Modules
Hero Strategy: A high-impact introduction focusing on core competencies in C++ 20, Python, and R, emphasizing a sub-10μs latency mindset.

Quantitative Portfolio: Displays a series of featured models, including a Monte Carlo VaR Engine, an Options Mispricing & Volatility Surface Engine, and a Market Regime Detection Engine.

Interactive UI Components: Includes a responsive mobile navigation system, animated terminal cursors, and CSS-driven hover effects designed to mimic financial dashboard interfaces.

Research Pipelines: Dedicated sections for secondary repositories, categorized by standard quantitative research and LLM-powered financial analysis.

Project Thesis
The portfolio is built on the guiding principle that "Code is the medium, mathematics is the message". It bridges the gap between theoretical stochastic calculus and practical algorithmic execution, providing a professional baseline for research collaborations and trading discussions.
