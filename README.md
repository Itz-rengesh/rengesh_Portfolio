# ⚡ RENGESH-QUANT | Engineering Alpha

> *"Code is the medium, mathematics is the message."*

Welcome to the source code for **RENGESH-QUANT**, a high-performance, single-page professional portfolio tailored for the quantitative finance industry. This repository serves as the centralized UI hub for displaying advanced algorithmic models, risk engines, and derivatives pricing frameworks.

---

## 🏗️ Technical Architecture

The front-end is engineered for speed and precision, utilizing a minimalist tech stack to ensure low-latency rendering and high readability:

* **⚙️ Framework:** Built with **Tailwind CSS v3**, leveraging a utility-first approach for responsive layouts and rapid component styling without the bloat.
* **🔤 Typography:** A dual-font system utilizing **Inter** for clean, modern headings and **JetBrains Mono** for technical/monospaced prose, reflecting a true developer environment.
* **🎨 Design System:** Features a bespoke dark-mode theme (`#0A0F14`) with a custom neon-green accent (`#00FF94`) and a CSS-generated geometric grid-overlay for a low-latency "terminal" aesthetic.

---

## 📂 Core Functional Modules

The UI is structured to highlight the bridge between theoretical stochastic calculus and algorithmic execution.

* **🚀 The Hero Strategy:** A high-impact introduction focusing on core execution competencies (**C++ 20**, **Python**, **R**) and a sub-10μs latency mindset.
* **📊 Quantitative Portfolio:** The primary display matrix for featured mathematical models, including:
    * 🎲 *Portfolio Risk & Monte Carlo VaR Engine*
    * 📈 *Options Mispricing & Volatility Surface Engine*
    * 🧠 *Market Regime Detection Engine*
* **🖥️ Interactive UI:** Includes a custom off-canvas mobile navigation system, CSS-animated retro terminal cursors (`>_`), and hardware-accelerated hover states.
* **🔗 Research Pipelines:** Dedicated directory grids for secondary repositories, neatly categorized by standard quantitative research and LLM-powered financial analysis.
